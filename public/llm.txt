# Better Flow - Free Screenshot Editor & Mockup Maker

## What is Better Flow?
Better Flow is a free, browser-based screenshot editor and mockup maker. It transforms plain screenshots into professional graphics in seconds — no signup, no watermarks, no cost. Users upload a screenshot, apply backgrounds, frames, effects, and animations, then export as PNG, JPG, WebP, MP4, WebM, or GIF.

## Core Features
- **100+ Backgrounds** - Gradients, mesh, radiant, solid colors, paper textures, and custom image backgrounds
- **Browser Mockups** - Safari and Chrome browser frames in light and dark mode
- **Device Frames** - macOS window, Arc browser, Polaroid, glass, outline, and border frames
- **3D Transforms** - Perspective tilt, rotation, and depth effects
- **Text & Image Overlays** - Custom fonts (30+ Google Fonts), colors, sizes, shadows, and positioning
- **Animation Timeline** - 20+ animation presets: zoom, pan, Ken Burns, tilt, fade, and custom keyframes
- **Video Export** - Export as MP4, WebM, or GIF using FFmpeg WASM, WebCodecs, or MediaRecorder
- **High-Resolution Export** - PNG, JPEG, WebP up to 5x scale
- **Presets** - One-click professional styling presets
- **No Signup Required** - All features available instantly

## Alternatives & Comparisons
Better Flow is a free alternative to Pika Style, Shots.so, Screely, and Cleanshot X's decoration features. Unlike paid tools, Better Flow has no watermarks, no subscription, and no export limits.

## Use Cases
- Beautifying screenshots for SaaS landing pages and product marketing
- Creating social media graphics for Twitter/X, LinkedIn, Instagram
- Generating developer portfolio and project showcase images
- Producing animated product demos and walkthroughs
- Making code snippet screenshots with stylized backgrounds

## Target Audience
Developers, designers, indie hackers, content creators, and product teams who need to create professional visual content quickly without design skills or paid software.

## Pricing
100% free. No hidden costs, premium tiers, usage limits, or watermarks. Unlimited exports.

## Technology
- Next.js 16 (App Router), React 19, TypeScript
- HTML Canvas + modern-screenshot for rendering
- FFmpeg WASM, WebCodecs, MediaRecorder for video encoding
- Cloudflare R2 for asset storage
- Firebase Authentication
- Zustand for state management
